the script is auto-split into cues. the cue editor (bottom right, or press C) holds them for editing — ↑ ↓ arrows trigger each one.

dialogues-only follows stage-script layout: speaker names on their own line, directions as plain paragraphs. if no structure is found, the full text is used.

re-splits the uploaded script with the setting above. hand edits to the cue list are replaced.

Translation

two-box mode places the original in the upper half and the translation in the lower half, so two crops can feed two screens. the safe-area box outlines each.

upcoming cues translate themselves a few ahead as you step. translate-all prepares the whole show beforehand — do it once on venue wifi, then export cues to keep the translations.

Characters

speaker names detected as NAME: at the start of lines. each character's text can carry its own style and colour. names found in the script appear here automatically.

Canvas

draws the maximum extent of the current cue list, so the crop in OBS or Isadora can be set once and every cue stays inside it. measure with the menus closed, then switch off for the show.

the canvas slowly darkens when no one speaks.
